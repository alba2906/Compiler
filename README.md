## Используемые технологии

C# 8, WPF, Visual Studio 2022

Инструкция по сборке и запуску
a. Установка зависимостей

Установить Visual Studio 2022.
При установке выбрать пункт "Разработка классических приложений .NET".
b. Команды для сборки проекта

Используя меню Visual Studio 2022: Сборка > Собрать решение.
c. Путь к готовому исполняемому файлу

После сборки исполняемый файл формируется в директории Compiler\GUI\bin\Release\net9.0-windows или LanguageProcessor\GUI\bin\Debug\net9.0-windows в зависимости от выбранной конфигурации.
Так же можно воспользоваться прекомпилированными версиями приложениями из вкладки Release

## Описание интерфейса и функций (руководство пользователя)
Интерфейс состоит из меню, содержащего все доступные операции, панели инструменов, дублирующей функционал меню, окна ввода текста и окна вывода диагностической информации.

Раздел "Файл":

Создать - создание нового документа (очистка области редактирования)
Открыть - загрузка существующего файла с диска
Сохранить - сохранение текущего документа
Сохранить как - сохранение документа с указанием нового имени/пути
Выход - завершение работы приложения (с проверкой несохраненных изменений)
Раздел "Правка":

Отменить - отмена последнего действия (Ctrl+Z)
Повторить - повтор отмененного действия (Ctrl+Y)
Вырезать - удаление выделенного текста в буфер обмена (Ctrl+X)
Копировать - копирование выделенного текста в буфер обмена (Ctrl+C)
Вставить - вставка текста из буфера обмена (Ctrl+V)
Удалить - удаление выделенного текста (Del)
Выделить все - выделение всего текста в документе (Ctrl+A)
Раздел "Справка":

Вызов справки - открытие окна справки (F1)
О программе - информация о версии, авторе, лицензии
